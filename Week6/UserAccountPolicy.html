J-AND-B user Account Policy

Each user account on the J-AND-B system has the following allowances:

  • Logging into the system is via SSH. Other types of connections are not allowed.
  • 50 Megabytes of home (~) storage space is provisioned
  • The user shell is bash
  • Along with bash and other general system commands, applications available for use include SSH, git, Alpine email client, and Lynx web browser

The password policy requires changes every 60 days, and can only be changed once per day. Complexty requirements include a password length of no less than six charactors.
